# Prolog Game

## Project Overview
This project is a Prolog-based game where users can engage in a text-based adventure. The game utilizes the power of Prolog's logical reasoning capabilities to create a dynamic gaming experience.

## Features
- **Interactive gameplay**: Users can explore different scenarios and make choices that affect the outcome.
- **Dynamic story**: The narrative changes based on user choices, providing a unique experience each time.
- **Rich character development**: Users can interact with various characters that have their own stories and dialogues.

## Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/11FaZe11/prolog-game-
   ```
2. Navigate to the project directory:
   ```sh
   cd prolog-game-
   ```
3. Ensure you have SWI-Prolog installed on your machine.
4. Load the game in SWI-Prolog:
   ```prolog
   ?- consult('game.pl').
   ```

## Usage
To start the game, run the following command in the Prolog interpreter:
```prolog
?- start_game.
```
Follow the prompts to navigate through the game.

## Architecture
The game is structured as follows:
- **Core Logic**: Contains the main game logic and rules.
- **Game State**: Manages the current state of the game and user progress.
- **User Interface**: Handles the interaction between the game and the player.
